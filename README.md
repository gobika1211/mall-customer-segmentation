# Mall Customer Segmentation using Clustering Algorithms

## About this project
This project segments mall customers into distinct groups based on their
age, annual income, and spending score using unsupervised machine learning.

## Algorithms used
- K-Means Clustering
- Hierarchical Clustering

## Tools and libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## What I did
- Loaded and explored the mall customers dataset
- Cleaned and preprocessed the data
- Applied K-Means clustering to identify customer groups
- Applied Hierarchical clustering and compared results
- Visualized the clusters using Matplotlib and Seaborn
- Derived insights to support targeted marketing strategies

## Key learnings
- Understood how clustering algorithms work on real-world data
- Learned how to choose optimal number of clusters using Elbow method
- Practiced data visualization and interpretation of results
